## 🔊 發音規則（極重要）

### 核心原則
**TTS（Web Speech API）對注音符號的處理：**
- 單一注音符號 → 老師端 TTS（Microsoft Hanhan/Yating）會念出該注音本身的音 ✅
- 兩個以上注音 → TTS 會拆開念，所以要改用代表合念音的中文字

### 三種發音策略

| 類別 | speech 欄位內容 | TTS 念出 | 範例 |
|---|---|---|---|
| **結合韻**（多注音組成一音節）| 一聲中文字 | 完整合念音 | ㄧㄢ → speech: `'煙'` → 念「yān」 |
| **單韻**（單一注音） | 注音符號本身 | 該韻的純音 | ㄢ → speech: `'ㄢ'` → 念「ㄢ」 |
| **聲符**（純子音） | 注音符號本身 | 該聲符的子音 | ㄅ → speech: `'ㄅ'` → 念「ㄅ」 |

## 🎨 視覺顯示規則

### 直式注音（vbopo）
- 所有注音符號採**直式顯示**（垂直堆疊）
- 結合韻例：ㄧㄢ 顯示為 `一` 在上、`ㄢ` 在下
- **ㄧ 維持橫線（一）形狀，不旋轉**
- 實作方式：`function vbopo(s)` 把字串拆成單字元，用 inline-flex column 堆疊
- CSS class：`.vbopo`、`.vbopo > span`

### 配色（兒童友善）
- 主色：粉紅 `#FFB6D9`、`#ff6b9d`（女孩友善）
- 副色：嫩黃 `#FFC857`、淡藍 `#A6D9F7`、嫩綠 `#7ED957`
- 背景：粉色到藍色到黃色的漸層

### 字體與按鈕
- 注音字大：認讀字卡 120px、按鈕 48px
- 按鈕圓潤、有圓角（border-radius: 16-20px）
- 答對動畫：彈跳 + 撒星星
- 答錯動畫：搖晃（不挫敗）

## 🐛 已解決問題備註

### 1. TTS 把結合韻拆開念
- **問題**：ㄧㄢ 被念成「ㄧ + ㄢ」分開的兩個音
- **解法**：speech 欄位改用一聲中文字（煙）代表合念音

### 2. TTS 把代表字念成完整字
- **問題**：「杯子的杯」念成 bēi zi de bēi（杯念成 ㄅㄟ）
- **解法**：聲符的 speech 直接用注音符號（如 `'ㄅ'`），讓 Windows Hanhan 念出純子音

### 3. ㄧ 在直式中的方向
- **使用者偏好**：橫線（一）形狀，不要旋轉成直線
- **目前實作**：vbopo() 不對 ㄧ 做任何特殊處理

### 4. 檔案被截斷
- **問題**：大檔案 Edit 操作時偶爾會截斷尾部
- **解法**：每次 Edit 後用 Python 檢查 `</script>` 與 `</html>` 是否存在
- **修復方法**：用 Python 重寫整段 PHONETIC_DATA，或附加缺失的尾部
